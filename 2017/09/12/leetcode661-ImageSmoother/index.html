leetcode#661 Image Smoother

Given a 2D integer matrix M representing the gray scale of an image, you need to design a smoother to make the gray scale of each cell becomes the average gray scale (rounding down) of all the 8 surrounding cells and itself. If a cell has less than 8 surrounding cells, then use as many as you can.

>

Note:

  1. The value in the given matrix is in the range of [0, 255].
  2. The length and width of the given matrix are in the range of [1, 150].

解释

给定一个二维整数矩阵M，用于表示图像的灰度，要求编写程序，计算每一个cell（数组元素值）的平均灰度，计算方法是：将围绕该点（相邻）的其他元素值以及自己本身的值加起来，然后求平均，然后舍去小数部分。

假设：

  • 元素值大小范围：[0,255]；
  • 矩阵的长度和宽度范围：[1,150]。

我的解法

本题考察点在于二维数组的边界，很直白，只要判定好边界，保证不越界，就可以加和在一起，连带的计数值也需要累加，最后将累加值除以计数值，得到平均数，放在结果数组的ij位置即可。

大神解法

思路是一致的。巧妙的地方在于，利用了一个计数值count 这样可以避免很多的条件判断语句。
